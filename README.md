🐍 Scripting-Python

A collection of Python scripting experiments focused on automation, file handling, and system-level tasks.

⚠️ WARNING

Do NOT run these scripts blindly.

Some scripts in this repository can:

Rename large numbers of files
Modify or move files
Potentially break file structures if misused
Overwrite existing data

Run only if you:

Understand what the script does
Have tested it in a safe folder
Have backups of important files
📌 About This Repo

This repository is for learning and experimenting with Python scripting.

It includes scripts that demonstrate:

File automation
Batch renaming
System operations using os
Basic scripting logic and workflows
🧠 Purpose

The goal of this repo is to:

Improve Python scripting skills
Understand real-world automation logic
Practice working with files and directories
Build foundational tools for automation
